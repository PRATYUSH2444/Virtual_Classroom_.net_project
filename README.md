# 🎓 Virtual Classroom System

This is a Virtual Classroom web application that I built as part of my semester project.
The main idea was to create a simple platform where teachers and students can manage classes, share content, and handle assignments in one place.

---

## 💡 What it does

* Teachers can create classrooms and manage them
* Students can join classrooms and access study material
* Assignments can be given and submitted
* Basic dashboards for Admin, Teacher, and Student
* File uploads for notes, assignments, etc.

---

## ⚙️ Tech Used

* ASP.NET Core MVC (.NET 8)
* SQL Server
* Entity Framework Core
* Razor Views + Bootstrap
* Visual Studio

---

## 📁 Project Structure (basic idea)

```bash
Controllers/   # handles logic  
Models/        # database tables  
Views/         # UI pages  
Data/          # DbContext  
Migrations/    # database updates  
wwwroot/       # static files + uploads  
```

---
